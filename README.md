## 图片检索系统

架构：tkinter

### 图片数据集：

Caltech-256

### 特征选择方式：
* 基于颜色直方图
* 基于方向梯度(HOG)
* 基于神经网络(VGG16)

### 检索方式：

* 线性检索
* K-Means聚类
* KD-tree
